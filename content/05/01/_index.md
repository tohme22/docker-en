### Exercice - Adding environment variables to a container

1- Download and run the **alpine** image. Use the following variables:

- `$LANG` --> Français (Canada) 
- `PS1` (Prompt) --> [root 18:19 ]$

2-  Create two containers:

- For the 1st container: Define the variables in the same docker command.

- For the 1st container: Define the variables in a file.

Note : The container should be automatically deleted once stopped.
